# Markdown Dokumentation

## Fordele ved at bruge Markdown
- Simpelt og hurtigt: Det er intuitivt og kræver kun få symboler (f.eks. stjerner til fed/kursiv), hvilket gør skrivningen hurtigere end traditionelle tekstbehandlingsprogrammer.
- Let at læse og skrive: Markdown-filer er rene tekstfiler (.md), der er menneskelæselige, selv uden formatering.
- Platformsuafhængig: Da det er ren tekst, kan filer åbnes i enhver tekst-editor på tværs af styresystemer.


## Her er et eksempel på et for-loop i JavaScript
```JavaScript

const cars = ["BMW", "Volvo", "Saab", "Ford"];
let len = cars.length;

let text = "";
for (let i = 0; i < len; i++) {
  text += cars[i];
}

```
